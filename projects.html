Line follower

Designed and built a line follower robot as part of a team using an ESP32 microcontroller with Wi-Fi connectivity. The robot utilizes 5 sensors for line detection, PID controller and computer vision to detect bends in the line, triggering an LED indicator. It boasts three operational modes: 1) Regular line following, 2) Maze solving with shortest path calculation and storage in EEPROM memory, and 3) Maze solving using the stored shortest path. Additionally, a custom PCB was designed to house the ESP32 and essential connectors.
